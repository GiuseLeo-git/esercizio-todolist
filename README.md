# Esami Universitari - Graphic & Web Design

Una moderna e funzionale applicazione per gestire gli esami universitari del corso di Graphic & Web Design, creata utilizzando esclusivamente HTML, CSS e JavaScript vanilla.

## 🚀 Funzionalità

### ✨ Funzionalità Principali
- **Aggiungere esami**: Inserisci nuovi esami tramite il campo di input
- **Superare esami**: Clicca sul checkbox per segnare un esame come superato
- **Modificare esami**: Clicca sull'icona di modifica per modificare il nome di un esame
- **Eliminare esami**: Clicca sull'icona del cestino per eliminare un esame
- **Filtrare esami**: Visualizza tutti gli esami, solo quelli da sostenere o solo quelli superati

### 🎨 Caratteristiche UI/UX
- **Design moderno**: Interfaccia pulita e accattivante con gradiente colorato
- **Animazioni fluide**: Transizioni e animazioni per un'esperienza utente migliore
- **Responsive**: Si adatta perfettamente a dispositivi mobili e desktop
- **Notifiche**: Feedback visivo per tutte le azioni dell'utente
- **Persistenza dati**: Le attività vengono salvate nel localStorage del browser

### 🔧 Funzionalità Avanzate
- **Contatore esami**: Visualizza il numero di esami da sostenere e superati
- **Eliminazione multipla**: Elimina tutti gli esami superati con un clic
- **Stato vuoto**: Messaggio informativo quando non ci sono esami
- **Tastiera**: Supporto per il tasto ESC per annullare la modifica
- **Validazione**: Controlli per evitare esami vuoti

## 🛠️ Come Utilizzare

### Installazione
1. Scarica tutti i file nella stessa cartella
2. Apri `index.html` nel tuo browser web
3. Inizia a utilizzare la To-Do List!

### Utilizzo Base
1. **Aggiungere un esame**:
   - Scrivi il nome dell'esame nel campo di input
   - Premi Invio o clicca il pulsante "+"

2. **Superare un esame**:
   - Clicca sul cerchio accanto all'esame

3. **Modificare un esame**:
   - Clicca sull'icona della matita
   - Modifica il nome dell'esame nel campo di input
   - Premi Invio per salvare

4. **Eliminare un esame**:
   - Clicca sull'icona del cestino

### Filtri
- **Tutti gli Esami**: Mostra tutti gli esami
- **Da Sostenere**: Mostra solo gli esami non superati
- **Superati**: Mostra solo gli esami superati
